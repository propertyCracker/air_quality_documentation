

# System Summary

The project consists of the following major components:

1. **Data Generation Module**: Simulates hourly air quality data for multiple locations.
2. **IoT Integration Module**: Streams the simulated data to Azure IoT Hub for ingestion.
3. **Data Storage Module**: Stores and organizes the incoming data in Azure Synapse Analytics.
4. **Analytics Module**: Uses Power BI to provide batch and streaming analysis of air quality data.
5. **LLM Module**: Allows users to query the data and receive insights through natural language commands.

This modular architecture ensures scalability, maintainability, and flexibility for future enhancements.
